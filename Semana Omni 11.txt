Além do npm que instala um pacote na nossa maquina no node tempos o 
NPX que executa pacotes externamente, por exemplo para criar um projeto react
podemos usar o pacote fornecido pelo facebook

>npx create-react-app NomeDoProjeto

Metodos HTTP:

GET: Busca/Listar uma informação do Back-end (Metodo padrão dos Browsers)
POST: Criar Um a informação no Back-end
PUT: Alterar uma informação no Back-end
DELETE: Deletar uma informação no Back-end

Tipos de Parâmetros

Query Param: Parâmetros nomeados enviados na rota após "?" e podem ser anexados por "&" (Filtros, Paginação)
Route Param: Parametros utilizados para identificar recursos (/:id)
Request Body: Corpo da requisição, utilizado para criar ou alterar recursos

Bancos de dados

SQL(Relacional): MySQL, SQLite, PostgreSQL, Oracle, Microsoft SQL Server
NoSQL (não Relacional): MongoDB, CouchDB, etc
